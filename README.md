## 📖 About DevVerse

**DevVerse** isn't just a portfolio — it's a self-contained universe documenting a developer's journey: skills, projects, milestones, and everything in between. Built with zero dependencies, it loads instantly, deploys anywhere, and is easy to customize without touching a single build tool.

> 💡 **Why "DevVerse"?** Because a developer's world spans more than just code — it's projects, growth, goals, and community. This site brings all of that into one connected universe.

## 🚀 Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/your-username/DevVerse.git
cd DevVerse
```

### 2. Run it locally
No build step needed — just open it:
```bash
# Option A: double-click index.html

# Option B: serve it properly (recommended)
python3 -m http.server 8000
```
Visit `http://localhost:8000` 🎉

### 3. Customize
| What to change | Where |
|---|---|
| Your name, bio, hero text | `index.html`, `about.html` |
| Profile photo, logo, resume | `assets/profile.jpg`, `assets/logo.png`, `assets/resume.pdf` |
| Colors & theme | CSS variables at the top of `assets/css/style.css` |
| Social links | `<footer>` section in each page |
| Project cards | `projects.html` + `project-details.html` |

## 🌍 Deployment (GitHub Pages)

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Source: `Deploy from a branch` → Branch: `main` → Folder: `/root`
4. Save — your site goes live at:

```
https://your-username.github.io/DevVerse/
```
